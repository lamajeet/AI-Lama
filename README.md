# AI-Lama
التعرف على الأرقام المكتوبة بخط اليد MNIST Handwritten Digit Recognition
يهدف هذا المشروع إلى بناء وتطوير نموذج ذكاء اصطناعي قادر على تصنيف الأرقام المكتوبة بخط اليد بدقة عالية. تم استخدام مجموعة بيانات MNIST الشهيرة، والتي تحتوي على صور لأرقام من 0 إلى 9. يركز المشروع على حل مشكلة التباين في الخطوط اليدوية وصعوبة التمييز بين بعض الأرقام المتشابهة مثل (3 و5) أو (4 و9)
التقنيات المستخدمة (Tech Stack)
تم بناء المشروع باستخدام لغة Python والاعتماد على المكتبات التالية:


NumPy & Pandas: لمعالجة البيانات والمصفوفات الرقمية.


Scikit-Learn: لتقسيم البيانات، وبناء نماذج التعلم الآلي، وتقييم الأداء.


TensorFlow & Keras: لبناء وتدريب الشبكات العصبية المتقدمة.


Matplotlib & Seaborn: لتمثيل البيانات بصرياً وعرض مصفوفة الالتباس (Confusion Matrix)
النتائج (Results)
حقق المشروع نتائج ممتازة تعكس تطور الأداء من البساطة إلى الاحترافية:
